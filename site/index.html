Remote desktop,
finally pleasant.

Waypoint is a modern RDP manager for Windows — tabs, standalone windows across multiple monitors, crisp 1:1 full screen and secure passwords. A comfortable replacement for the built-in Remote Desktop Connection (mstsc).

Features

Everything missing from mstsc

Built on Microsoft's official RDP control (mstscax) — full compatibility, with a modern Fluent / Mica interface on top.

Tabs and standalone windows

Multiple sessions in one window, or each in its own — whatever suits you. Duplicate and reorder tabs.

Multi-monitor, mstsc-style

Tear a tab off into its own window, drag it to another screen and go full screen. Dock it back with one click.

Crisp 1:1 full screen

Dynamic resolution matched to the monitor (DPI-aware) — no blur, with an auto-hiding bar (pin / minimize).

Secure passwords

Credentials kept in Windows Credential Manager (DPAPI) — never in app files.

Import / export .rdp and profiles

Move over from mstsc in a minute. Export your whole server and settings profile to a single file.

Redirections and RD Gateway

Clipboard, drives, printers, audio, authentication level and RD Gateway — configurable per server.

Security

Your passwords stay with you

Waypoint never writes passwords to its own files. They go to Windows Credential Manager (DPAPI, tied to your account) — the same vault mstsc uses. Server-identity verification (MITM protection) is on by default and configurable per server.
